# Blogverse Backend

This project is the backend for the Blogverse application, a Medium-like blogging platform, built with Rust and the Actix-web framework.

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install)

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd blogverse-backend
    ```

2.  **Build the project:**
    ```bash
    cargo build
    ```

3.  **Run the server:**
    ```bash
    cargo run
    ```
    The server will start on `http://127.0.0.1:8080`.

## API Endpoints

- `GET /health`: Health check endpoint to verify the service is running.
- `GET /user`: A placeholder endpoint.
